# online-store-recommendation-ab-test
Análisis completo del experimento A/B “recommender_system_test” que evalúa un nuevo sistema de recomendaciones en una tienda en línea. Incluye validación del diseño experimental, análisis de eventos, revisión del calendario de marketing para evitar sesgos, evaluación del funnel y pruebas estadísticas para determinar el impacto del sistema propuesto
